# Swipemoji

An iOS keyboard extension that inputs gestures for emojis.

# Key Features
  ##Requirements
  - Inputting gestures
  - Recognizing and interpreting gestures
  - Outputting emoji into text field
  
  ##Optionals
  - Display top four most related
  - Add other characters like alpha and numeric
  - Nueral net learning
